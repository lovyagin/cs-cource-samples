# dynamic_array

Иллюстрация создания динамического массива с динамической вместимостью

Поддерживается массив целых чисел

## Сборка и запуск (GNU/Linux)

### Сборка

```
make 
```

### Сборка и запуск тестов

```
make test
```

### Для очистки

```
make clean
```

### Valgrind

```
valgrind ./int_array_test
```

### Документация

```
make docs
```
